None beyond backend-connection.

## 6. Edge cases & error handling
- Duplicate keyed on `(organization_id, feedback_id)`, not content; `force` overrides.
- Feedback not owned by org → 404 (no info leak).
- Auth error mid-create → 403 + record `last_error`; token untouched (no rotation).
- Transient (429/5xx) → surfaced as 502, not retried in-process.
- `permalink_url` must be populated (opt_fields / fallback) or the "view in Asana" link breaks.

## Locked upstream contract (from aspect 1)

- `build_timeline(db, org_id, email, before=None, limit=20) -> (events, next_cursor)` in
  `src/services/customer_timeline_service.py`.
- `ActivityEvent` (Pydantic) + `TimelineResponse{events, next_cursor}` in `routes/customers.py`.

## Phases (TDD)

- **Goal:** one function builds the Customer 360 profile dict from a `CustomerHealth` row so v1
  `CustomerProfileResponse` and the public profile can't diverge.
- **Steps:** RED test asserting the serializer returns all profile fields (health_score, risk_level,
  confidence, 5 components incl. usage, churn_probability/bucket, llm summary fields, feedback_count,
  last_feedback_at). Extract the existing v1 profile-building logic
  (`routes/customers.py` `get_customer_profile`, ~`:325-398`) into

  have the v1 route call it. Confirm v1 profile tests still pass (characterization).
- **Validation:** `pytest tests/test_customer_profile.py -v`.
- **Commit:** `refactor(customers): extract shared Customer 360 profile serializer`

### Phase 2 — Public `GET /api/public/v1/customers/{email}`

- **Validation:** `pytest tests/ -k public -v`.

### Phase 4 — Extend `GET /api/public/v1/customers/{email}/health` (if thin)

  GREEN, ensure old assertions still hold.
- **Commit:** `feat(public-api): include component breakdown in customer health endpoint`

## Acceptance criteria (from spec)

1. Each endpoint 200 with documented shape for an existing customer (valid read key).
2. Missing `read` scope → 403; bad/no key → 401.
3. Cross-org → 404, never other org's data.
4. Public profile fields == v1 profile for the same customer (shared serializer; AC parity test).
5. Public timeline paginates identically to v1.
6. Unknown email → 404 on all.

## Edge cases & multi-tenancy

- Org from `auth.organization_id` on EVERY query; cross-org tests mandatory.
- `before` malformed → 422 (reuse v1 validation path).
- Decide at review: redact LLM free-text in public profile? Default NO (operator's own self-hosted
  data) — implement as-is, leave a TODO note; do not gate behind anything.

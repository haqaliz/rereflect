## 6. Edge cases & error handling

- Enabling when no HubSpot integration exists ŌåÆ 404/400 (mirror existing `/status` when disconnected).
- Field name with leading/trailing spaces ŌåÆ trim before validate/persist.
- Validation transient (429/5xx) during PATCH ŌåÆ surface a retryable 503-style message; do not persist enabled.
- Downgrade must not fail if columns already dropped (guard).

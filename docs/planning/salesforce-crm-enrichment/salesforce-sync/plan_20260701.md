## Phase 1 — Salesforce client (mocked HTTP)
- **Files:** `services/worker-service/src/clients/salesforce.py` (new); `tests/test_salesforce_client.py` (new).
- **Steps (mirror `clients/hubspot.py` shape):**
  1. `SalesforceTransientError` (retryable 429/5xx).
  2. `SalesforceClient(refresh_token, instance_url, login_base, api_version)` context manager.
  3. `_refresh()` → POST `{login_base}/services/oauth2/token` (grant_type=refresh_token) → access_token; raise a non-transient auth error on `invalid_grant`.
  4. `query(soql)` → GET `{instance_url}/services/data/{api}/query?q=` ; follow `nextRecordsUrl`; `PER_RUN_PAGE_CAP` with WARNING when hit; read `Sforce-Limit-Info` header (log remaining; stop near zero).
  5. Typed helpers: `list_contacts()`, `get_account(id)`, `get_open_opportunities(account_id)` (SOQL: `SELECT Id,Name,StageName,Amount,CloseDate,IsClosed FROM Opportunity WHERE AccountId=:id AND IsClosed=false`).
- **Tests (RED):** pagination follows nextRecordsUrl; 429→transient; invalid_grant→non-retry auth error; limit header parsed.
- **Commit:** `feat(salesforce): REST/SOQL client (refresh, pagination, limit-aware)`

## Phase 2 — sync core `_sync_org` (RED→GREEN)
- **Files:** `services/worker-service/src/tasks/salesforce_sync.py` (new); `tests/test_salesforce_sync.py` (new).
- **Steps (mirror `hubspot_sync.py:161-273`):**
  1. Known-email set from `CustomerHealth`.
  2. Pull contacts; match by lowercased `Email`.
  3. Per match: resolve Account via `Contact.AccountId` (direct FK — no associations) → company/ARR (`Account.AnnualRevenue`, `Account.Type`→lifecycle).
  4. `_pick_renewal_opportunity(opps)`: not closed, has `CloseDate`, highest `Amount` → renewal_date/deal_name/deal_stage/deal_amount.
  5. `_upsert_enrichment(db, org_id, email, fields, now, provider="salesforce")` (Python upsert on `(org,email)`, SQLite-safe; sets `provider`).
  6. `_call_update_health(org_id, email, db)` (guarded import, mirror `hubspot_sync.py:136-153`).
- **Tests:** synthetic mocked SOQL → upsert row `provider='salesforce'` with correct fields; renewal pick correctness; idempotent re-run; health recompute once per match (mock).
- **Commit:** `feat(salesforce): _sync_org — email match, account/opp mapping, upsert(provider), health recompute`

## Phase 3 — task body + fan-out
- **Steps (mirror `hubspot_sync.py:281-403`):**

  2. `sync_all_salesforce` (`@shared_task name="src.tasks.salesforce_sync.sync_all_salesforce"`): fan-out over active integrations, per-org failures isolated.
  3. `sync_salesforce_org` (`@shared_task(bind=True, max_retries=3, default_retry_delay=30, name="src.tasks.salesforce_sync.sync_salesforce_org")`).
- **Tests:** transient→retry; invalid_grant→disconnect no-retry; missing key→non-retry; fan-out isolates failures.
- **Commit:** `feat(salesforce): retryable per-org task + fan-out (invalid_grant → disconnect)`

## Comparison Post Template

Each "Rereflect vs X" comparison post should follow this structure:

1. **Intro** — What problem both tools solve, why someone is comparing them
2. **[Competitor] overview** — Fair, accurate description (no straw-manning)
3. **Rereflect overview** — Honest positioning
4. **Feature comparison table** — Side-by-side on 8-10 dimensions
5. **Pricing comparison** — Transparent pricing for both
6. **When to choose [Competitor]** — Be honest about their strengths
7. **When to choose Rereflect** — Our differentiators
8. **Verdict** — Balanced recommendation based on use case

**Key differentiators to highlight across all comparison posts:**
- AI-powered automatic analysis (not just collection/voting)
- Sentiment analysis + pain point detection + urgency flagging in one tool
- Multi-channel ingestion (Slack, Intercom, email, CSV)
- AI Copilot for natural language queries over feedback data
- Simpler setup, faster time-to-insight
- Transparent pricing, generous free tier
